# pacli-environment
This utitility let you working in the "pacli enviroment", i.e. basically you don't have to type "pacli" before any pacli command.

Installation:
- download `pacli_env`, `pre_pacli_env` and `pacli_worlds` into the folder you are going to use `pacli_env` from;
- in the shell move into that folder;
- `chmod 755 pacli_env; chmod 755 pre_pacli_env`;
- `pacli_env`;
- enjoy.

Optionally you can create a symlink in your `/bin` folder to where `pacli_env` is in, to run `pacli_env` from whatever position your shell is pointing to.
